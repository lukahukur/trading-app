## Run Locally


First, you should create postgreSQL db and migrate schema.prisma

Then, install all of the dependencies for server 
```bash
  cd server
  npm i
```

Also, for the client

```bash
  cd client
  npm i
```

to start the server run

```bash
  npm run start:dev
```
for client

```bash
  npm run dev
```
